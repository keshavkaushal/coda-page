Overview

Features

Highlight the Next Keys

In Coda, the next keys to play will light up in red so you know exactly where to place your fingers. Finding chords just became so much easier!

Android, iOS and Windows

Coda runs directly in your web browser, so you don't need to install any apps or software. This also means it can run on any device. Android, iOS and Windows are all supported!

Set Song Bookmarks

Coda has useful controls to help you navigate through the song. For example, you can set a bookmark at a specific point. Whenever you want to rewind to that part, just tap once on the left-hand side of the screen.

Unlimited Song Library

Piano MIDI files are available online for almost every popular song. In addition, MIDI files for all 200+ piano covers on my YouTube channel are available for free!

Why Coda?

Coda was designed with simplicity in mind. It offers all of the core features of Synthesia for a fraction of the price. This makes it the ideal choice for the pianist who wants to learn new songs visually, but doesn't need the extra features of Synthesia.
