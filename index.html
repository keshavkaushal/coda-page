Features

New! Key highlighting ✨

Finding chords just became much easier! In Coda, the next keys to play will light up in red so you know exactly where to put your fingers. No more guessing!

Set useful markers 📌

Learning a new song requires a lot of repetition. Coda has useful touch controls to help you navigate through the song. For example, you can set a bookmark at a specific point. Whenever you want to rewind to that part, just tap once on the left of the screen.

Practice anywhere 🌎

Coda runs in your web browser, so you don't need to install any apps or software. This also means it can run on any device. Android, iOS and Windows are all supported!

Accurate piano sounds 🎹

Enjoy high-quality audio playback sampled from a Yamaha C5 grand piano.

Why Coda?

Coda was designed with simplicity in mind. It offers all of the core features of Synthesia for a fraction of the price. This makes it the ideal choice for the pianist who wants to learn new songs visually, but doesn't need the extra features of Synthesia.
